## How to Run

Follow these steps to run the project locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Soorya-Narayan/ASCII.git
    cd ASCII
    ```

2.  **Install dependencies**:
    Make sure you have [Node.js](https://nodejs.org/) installed, then run:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```

4.  **Open in browser**:
    Click the link provided in your terminal (usually `http://localhost:5173`) to view the application.
